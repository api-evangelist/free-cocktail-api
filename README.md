# Free Cocktail API (free-cocktail-api)
The Free Cocktail API (TheCocktailDB) provides access to a vast database of cocktail recipes, ingredients, glassware, and images. Search by name, ingredient, category, glass, or alcohol content; look up details by ID; list available filters; or fetch a random cocktail.

## APIs

### Free Cocktail API
The Free Cocktail API provides access to a vast database of cocktail recipes, ingredients, and images. The base URL is `https://www.thecocktaildb.com/api/json/v1/{apiKey}`; use API key `1` for development. Endpoints include `/search.php`, `/lookup.php`, `/random.php`, `/filter.php`, and `/list.php`.

**Human URL:** [https://www.thecocktaildb.com/api.php](https://www.thecocktaildb.com/api.php)

**Base URL:** `https://www.thecocktaildb.com/api/json/v1`
